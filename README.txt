Das Projekt SrM enthält die einfache Simulierte Register Maschine.

Das Projekt SrM-Updater wird davon eingebunden um Updates zu ermöglichen, ist aber unter Umständen nicht auf dem neuesten Stand da sich der Pfad auf github ändern sollte.

Das Projekt kann meinetwegen gerne kopiert, verändert und weitergenutzt werden, ich übernehme aber keine Verantwortung für den Funktionsumfang da ich dieses Projekt seit meiner Schulzeit nicht mehr weiterentwickle.
