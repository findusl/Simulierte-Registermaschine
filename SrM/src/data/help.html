Simulierte Registermaschine

Diese Jar-Datei stellt eine Entwicklungs- und Ausfhrungsumgebung fr Assemblercode zur Verfgung. Die verfgbaren Befehle sind:

Befehl

Auswirkungen

DLOAD i

Ldt unmittelbar die Zahl i in A, erhht den Wert in BZ um 1

LOAD x

Kopiert den Wert in Rx nach A, erhht den Wert in BZ um 1

STORE

Kopiert den Wert in A nach Rx, erhht den Wert in BZ um 1

ADD x

Addiert den Wert in Rx zum Wert in A, legt das Ergebnis in A ab, erhht den Wert in BZ um 1

SUB x

Subtrahiert den Wert in Rx vom Wert in A, legt das Ergebnis in A ab, erhht den Wert in BZ um 1

MULT x

Multipliziert den Wert in Rx mit dem Wert in A, legt das Ergebnis in A ab, erhht den Wert in BZ um 1

DIV x

Dividiert den Wert in A durch den Wert in Rx (Ganzzahldivision ohne Rest), legt das Ergebnis in A ab, erhht den Wert in BZ um 1

JUMP n

Ldt die Zahl n in BZ, Programm wird mit Befehl in Speicherzelle n fortgesetzt (Unbedingter Sprung)

JGE n

Ldt die Zahl n in BZ, falls der Wert in A grer oder gleich 0 ist, erhht ansonsten BZ um 1 (Bedingter Sprung)

JGT n

Bedingter Sprung grer als 0

JLE n

Bedingter Sprung kleiner oder gleich 0

JEQ n

Bedingter Sprung kleiner als 0

JEQ n

Bedingter Sprung gleich 0

JNE n

Bedingter Sprung ungleich 0

END

Erhht den Wert in BZ um 1 und beendet den Programmlauf
